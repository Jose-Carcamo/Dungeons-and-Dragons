# Dungeons-and-Dragons

A terminal-based dungeon escape game written in C++.

## How to Play
- Navigate a 5x5 grid using W/A/S/D keys
- Collect coins to increase your score
- Avoid traps or lose health
- Reach the exit (E) before your health runs out

## Game Symbols
- P = Player
- C = Coin
- T = Trap
- X = Wall
- E = Exit

## How to Run
g++ main.cpp -o dungeon
./dungeon

## Concepts Used
- Object-oriented programming (Player class)
- 2D arrays and grid traversal
- Boundary and collision detection
- Enums and switch statements
